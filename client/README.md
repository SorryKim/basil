## 'yarn install'
먼저 필요한 패키지들을 설치해줍니다.

## `yarn start`
이걸로 react 앱 실행
Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## `nodemon src/server.js`
터미널 하나 더 열어서 서버를 실행
Runs the Server
Open [http://localhost:8080](http://localhost:8080) to view it in your browser.
